Proxy Testing

To test WebWorks proxies please see the follow wiki link for a guide on using the WebWorks BDS environment and setting up proxy settings for multiple scenarios Wiki.

Test Steps

  • 1. Navigate to the Connections and setup the wifi proxy on device using the following credentials:
    • URL: bbsproxy3.bbs.testnet.rim.net
    • Port: 8080
  • 2. Try two scenarios both with, and without username and password, where you enter only the server and port.
  • 3. Open one of the pages below. If no username was entered you should be prompted by WebWorks to fill in the proxy details. If you entered a username or pass you should not be prompted for credentials.
  • 4. Verify that you are able to navigate to the pages below and that your credentials are saved and stored by the system.
  1. Twitter
  2. Pinterest
  3. RIM Internal
